# Scripts Directory

This directory contains utility scripts for maintenance and setup tasks.

## Contents:

- `generate_gmail_token.py` - Generate Gmail OAuth token for email sending
- `migrate_*.py` - Database migration scripts
- `fix_*.py` - Scripts to fix specific issues
- `patch_*.py` - Patch scripts for updates
- `show_llm_settings.py` - Display LLM settings

## Usage:

Run scripts from the backend root directory:
```bash
python scripts/<script_name>.py
```
